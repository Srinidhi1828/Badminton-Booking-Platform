Badminton Booking Platform:

The Badminton Booking Platform is a modern web application designed to make court reservations simple, fast, and user-friendly. It allows players to book badminton courts, rent equipment, schedule training sessions with professional coaches, and view all their upcoming or past bookings in one place. Built with a clean interface and smooth user flow, the platform focuses on delivering a seamless digital experience for both casual users and regular players.

This project is created using React (Vite) for the frontend and integrates with APIs for booking, pricing, and availability checks. The goal is to replace the outdated manual booking process with a smarter, automated system that reduces friction and saves time.

Key Features:
1. Court Booking

Users can select their preferred date, start time, end time, and court type. The UI automatically updates pricing and availability in real time, ensuring clarity before confirming a booking.

2. Equipment Rental

Players can choose from a list of available equipment such as rackets, shuttlecocks, and shoes. Quantity selection is supported, and the cost is automatically added to the total breakdown.

3. Coach Selection

Users can optionally book a certified coach for their training session. Fees are integrated directly into the pricing system for transparency.

4. Dynamic Price Breakdown

A separate card displays a complete price estimate:

Base court fee
Equipment charges
Coach fee
Surcharges
Multipliers
Final total

This helps users understand exactly what they are paying for.

5. Booking Management (My Bookings)

Users can view all past and upcoming bookings stored locally. The interface is clean and simple, showing date, time, and booking details, along with an option to cancel.

6. Fully Responsive Design

The entire application adapts to mobile, tablet, and desktop screens, ensuring a consistent and enjoyable experience.

Purpose of the Project:

The primary objective is to digitize and streamline the badminton reservation experience. Sports centers often rely on phone calls or paperwork, which are inefficient and prone to errors.
